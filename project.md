# Shift Organization

Shift Organization is a backend project that meets the stack requirements listed below. It's an MVP to refine the
concepts and prepare us for a technical interview. The real project on which we based this MVP covers topics such as:

* property management platform
* booking conflicts
* recurring events
* workflows
* complex business logic

## Role

We're looking for a Backend Developer to build and maintain scalable APIs for our property management platform. You'll
work with a modern Kotlin-based microservices architecture handling real-time operations. Collaborate on code reviews
and architecture decisions. Monitor and optimize API performance. Strong SQL/NoSQL database design
